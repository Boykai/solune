## Summary

The Solune codebase has extensive chat features — streaming SSE, Microsoft Agent Framework integration, @mention pipeline selection, file uploads with transcript detection, voice input, chat history navigation, AI Enhance toggle, issue recommendations, and markdown rendering — that are undocumented or only partially documented. This plan creates 1 new dedicated chat page guide (`docs/pages/chat.md`) and updates 5 existing documentation files (`api-reference.md`, `architecture.md`, `project-structure.md`, `roadmap.md`, `pages/layout.md`) plus the pages index (`pages/README.md`). No code changes are required — this is a documentation-only effort.
