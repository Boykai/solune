# Implementation Plan: Remove Fleet Dispatch & Copilot CLI Code

**Branch**: `002-remove-fleet-dispatch` | **Date**: 2026-04-13 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/002-remove-fleet-dispatch/spec.md`

## Summary

Remove all fleet dispatch orchestration and GitHub Copilot CLI plugin code from Solune (~30 files deleted, ~15 files modified). The existing "classic" dispatch path — which uses `format_issue_context_as_prompt()` and standard Copilot assignment via `assign_copilot_to_issue()` — becomes the sole execution path. Research confirmed all dependencies are internal, no external consumers exist for removed API fields, and the legacy pipeline stages fallback is the current production default.
