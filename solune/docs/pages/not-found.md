# Page Not Found (404)

If you navigate to a URL that does not exist in Solune, you will see the "Lost Between Sun & Moon" error page.

## What You See

- A **404** indicator
- A message explaining the page could not be found
- A **Go Home** button that takes you back to the Dashboard

## What to Do

- Click **Go Home** to return to the Dashboard.
- Alternatively, use the **sidebar** on the left to navigate to any valid page — the sidebar and top bar are still visible on this screen.
- If you arrived here from a bookmark or shared link, the page may have been moved or removed. Try navigating from the sidebar instead.
