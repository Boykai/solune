# Login

The Login page is the first screen you see when you open Solune. It provides a simple way to sign in with your GitHub account.

## What You See

- A full-screen welcome page with Solune branding
- A **Sign in with GitHub** button in the center
- A theme toggle (sun/moon icon) to switch between light and dark mode before signing in
- An animated background with celestial effects

## How to Sign In

1. Click the **Sign in with GitHub** button
2. You are redirected to GitHub's authorization page
3. Authorize Solune to access your GitHub account
4. GitHub redirects you back to Solune, and you are signed in

If you were trying to reach a specific page before being redirected to login (e.g., you bookmarked the Pipeline page), Solune remembers that and takes you there automatically after sign-in.

## Tips

- You can set your preferred theme (light or dark) before signing in — your choice carries over into the app.
- If you are already signed in and visit this page, you are taken straight to the Dashboard.
- Solune uses GitHub OAuth, so your GitHub password is never shared with Solune directly.
