# Help

The Help page is your in-app documentation center. It brings together getting-started guides, frequently asked questions, feature walkthroughs, and a reference for chat slash commands — all without leaving Solune.

## What You See

### Getting Started

A set of cards that introduce the core workflows in Solune. These are ideal if you are new and want a quick orientation on how to manage projects, set up pipelines, and work with agents.

### FAQ

An accordion of frequently asked questions organized by topic:

- **Getting Started** — basic setup and first steps
- **Agents & Pipelines** — how AI agents work, how to configure workflows
- **Chat & Voice** — using the chat panel, voice input, slash commands
- **Settings & Integration** — configuring Signal, AI models, and preferences

Click any question to expand its answer. Click again to collapse it.

### Feature Guides

A grid of cards, one for each major feature in Solune. Each card includes a brief description and links directly to the relevant page (Projects, Pipeline, Agents, Tools, Chores, Settings, Apps, Activity, etc.).

### Slash Commands

A table listing all available slash commands you can type in the chat panel:

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/theme` | Toggle light/dark mode |
| `/clear` | Clear chat history |
| ... | And more — the table always reflects the latest registered commands |

### Replay Tour

A button at the bottom of the page that restarts the onboarding tour. The tour walks you through the key UI elements step by step with spotlight highlights. Use this if you skipped the tour initially or want a refresher.

## How to Use It

- **New to Solune?** Start with the Getting Started cards, then scan the FAQ for your specific questions.
- **Looking for a specific feature?** Check the Feature Guides grid — each card links directly to the page it describes.
- **Forgot a slash command?** Scroll to the Slash Commands table for the full reference.
- **Want the guided tour again?** Click "Replay Tour" at the bottom.

## Tips

- The Help page works even if the backend is unavailable — all content is built into the app.
- The slash commands table stays up to date automatically as new commands are added to the chat system.
